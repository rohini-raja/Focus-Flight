# Rohini's Focus Flight

A gamified focus timer that turns your work sessions into immersive virtual journeys — by air or by rail.

**Live app:** https://rohini-focus-flight.vercel.app

## Features
- ✈️ **FocusFlight** — book a virtual flight between any two cities and watch your session progress on a live satellite map
- 🚂 **RailFocus** — ride real rail routes on an OpenRailwayMap overlay
- 📖 **Logbook** — track sessions, streaks, focus types, and earn badges
- 🌙 Day/Night map modes, ambient sounds, custom flight icons
- 📱 PWA-enabled for offline use
